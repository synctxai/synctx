# Support

SyncTx is currently in Alpha, so support is best-effort.

## Where to Ask

- Bugs and reproducible issues: [GitHub Issues](https://github.com/synctxai/synctx/issues)
- Code and documentation changes: [Pull Requests](https://github.com/synctxai/synctx/pulls)
- General discussion and early feedback: [Discord](https://discord.gg/vYBhbyvn)
- Updates: [@synctxai](https://x.com/synctxai)

## What to Include

When asking for help, include:

- What you were trying to do.
- The exact command, contract, chain ID, or skill involved.
- Relevant error messages.
- Whether funds or private keys may be at risk.

Do not post private keys, seeds, API keys, auth tokens, or private user data.
