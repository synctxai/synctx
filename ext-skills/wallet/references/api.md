# Wallet Skill API Reference

## wallet.py ÔÇö Account & Utilities

| Function | CLI Command | Returns | Description |
|----------|-------------|---------|-------------|
| `address()` | `address` | `str` | Wallet address (local only, no RPC) |
| `eth_balance(chain_id=10)` | `eth-balance` | `dict` | `{address, chain_id, balance_raw, balance}` |
| `to_raw(amount, decimals=18)` | `to-raw` | `int` | Human-readable to raw (e.g. 1.5 USDC Ôćĺ 1500000) |
| `fmt(raw, decimals=18, symbol="")` | `fmt` | `str` | Raw to human-readable (e.g. 1500000 Ôćĺ "1.5 USDC") |
| `sign_message(message)` | `sign-message` | `dict` | EIP-191 signature `{address, message, signature}` |
| `sign_typed_data(typed_data)` | `sign-typed-data` | `dict` | EIP-712 signature `{address, signature, v, r, s}` |

## abi.py ÔÇö Contract Interaction

| Function | CLI Command | Returns | Description |
|----------|-------------|---------|-------------|
| `list_functions(abi_or_address)` | `list-functions` | `dict` | `{read: [sig...], write: [sig...]}`, auto-fetches ABI for contract addresses |
| `call(address, sig, args, *, chain_id, from_address)` | `call` | auto | Read operation, return value auto-serialized |
| `invoke(address, sig, args, *, chain_id, value)` | `invoke` | `dict` | **Only write entry point**, returns serialized receipt (tx_hash as hex string), auto-decodes revert errors |

## erc20.py ÔÇö ERC20 Convenience Operations

| Function | CLI Command | Returns | Description |
|----------|-------------|---------|-------------|
| `balance(token, owner=None, *, chain_id)` | `balance` | `dict` | `{raw, formatted, symbol, decimals}`, owner defaults to current wallet |
| `approve(token, spender, amount, *, chain_id)` | `approve` | `dict\|None` | Checks allowance, approves if insufficient; returns None if already sufficient |
| `approve_and_invoke(token, contract, amount, sig, args, *, chain_id, value)` | `approve-and-invoke` | `dict` | Approve + invoke in one step |

## decoder.py ÔÇö Error & Event Decoding

| Function | CLI Command | Returns | Description |
|----------|-------------|---------|-------------|
| `decode_logs(tx_hash, contract_address, *, chain_id)` | `decode-logs` | `list[dict]` | Parse transaction event logs |
| `decode_revert(error_data, contract_address, *, chain_id)` | `decode-revert` | `str` | Decode revert reason (usually not needed manually ÔÇö invoke auto-decodes) |

## sig Format

sig includes return type: `"balanceOf(address)->(uint256)"`, generated by `list-functions`.

Automatic argument type conversion:
- `address` Ôćĺ checksum format
- `uint/int` Ôćĺ integer
- `bool` Ôćĺ `true`/`false`
- `bytes` Ôćĺ hex string
- `tuple/array` Ôćĺ JSON parsed

**Do not modify sig format. Do not embed argument values in sig.**

## CLI Conventions

All CLI commands output JSON to stdout; diagnostics go to stderr.

Common options:
- `--chain <id>` ÔÇö Chain ID, default 10 (Optimism)
- `--args '<json_array>'` ÔÇö Function arguments as JSON array, e.g. `'["0x...", "1000"]'`
- `--dry-run` ÔÇö Estimate gas only, do not execute (`invoke` command only)
- `--help` ÔÇö Show command help

Exit codes:
- `0` ÔÇö Success
- `1` ÔÇö Bad arguments
- `2` ÔÇö Contract revert (stderr contains decoded revert reason)
- `3` ÔÇö Network/RPC error
- `4` ÔÇö Private key not configured (.env missing or PRIVATE_KEY not set)

Error output format (stderr JSON):
```json
{"error": "ContractRevert", "detail": "Transaction would revert: InsufficientBalance(required=1000000, available=500000)", "exit_code": 2}
```

## `-c` Advanced Mode

For composing multi-step operations or scenarios not covered by subcommands:

```bash
uv run scripts/run.py -c "from erc20 import approve_and_invoke; print(approve_and_invoke(...))"
```

In `-c` mode you can directly import all internal modules (wallet, abi, erc20, decoder) and compose freely in Python. Prefer subcommand mode; use `-c` only when you need multi-step composition.

## Utility Function Examples

```bash
# Human-readable amount Ôćĺ raw integer
uv run scripts/run.py to-raw 1.5 --decimals 6
# Output: 1500000

# Raw integer Ôćĺ human-readable
uv run scripts/run.py fmt 1500000 --decimals 6 --symbol USDC
# Output: "1.5 USDC"

# Manually decode revert data (usually not needed ÔÇö invoke auto-decodes)
uv run scripts/run.py decode-revert 0x08c379a2... --contract 0x... --chain 10
```
